# Ciao, sono Sara 👋
 
Studentessa di Economia Bancaria e Finanziaria, particolarmente interessata al mondo del **Data Analysis** e del **Financial Engineering**.
 
Vorrei costruire il mio percorso professionale nella data analytics, e per questo sto imparando da zero, costruendo progetti uno dopo l'altro per esercitarmi e padroneggiare gli strumenti del settore: SQL, Python, Power BI ed Excel.
 
## 📂 I miei progetti
 
- 🛍️ **Analisi Vendite E-commerce** — SQL, Python e Power BI su dati di vendita
- 👥 **Analisi HR** — query su un database con dipendenti, stipendi e dipartimenti
- 💳 **Rilevamento Frodi Finanziarie** — analisi di transazioni su larga scala con Python
- 🏦 **Customer Churn Bancario** — identificazione dei fattori di abbandono clienti, con proposte di azione concrete
